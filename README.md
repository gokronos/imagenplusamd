# Imagen Plus

Sitio profesional de la agencia Imagen Plus, construido con Next.js 15, React 19,
TypeScript, Tailwind CSS y Sanity CMS.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run lint
npm run typecheck
```

## Variables de entorno

Copia `.env.example` a `.env.local` y completa las credenciales de Sanity antes
de consumir contenido real.
